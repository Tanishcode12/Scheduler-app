# Scheduler-app
Android task scheduler built with Kotlin, Jetpack Compose, and Room. It supports creating tasks with priorities, categories, and due dates, plus search, filtering, pinning, swipe-to-delete, and a calendar view. All data is stored locally for fast, offline access.
# Features
- Create, edit, and delete tasks
- Pin important tasks to the top
- Mark tasks as completed
- Swipe to delete with undo support
- Search and filter tasks by category
- Custom categories and priority levels (Low, Medium, High)
- Calendar view for tasks by date
- Due date and time selection
- Persistent local storage using Room Database
# Tech Stack
- Language: Kotlin
- UI: Jetpack Compose (Material 3)
- Architecture: MVVM
- Database: Room (SQLite)
- State Management: ViewModel + State
# Privacy
This app uses a local Room database.
No user data is collected, stored externally, or transmitted.
